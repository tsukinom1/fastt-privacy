1. Общая информация

Приложение Fastt ("мы", "нас", "наше") уважает вашу конфиденциальность и обязуется защищать ваши личные данные. Данная политика конфиденциальности описывает, как мы собираем, используем и защищаем информацию, которую вы предоставляете при использовании нашего приложения для зарядных станций.

2. Собираемая информация

2.1 Информация, которую вы предоставляете:

  • Данные для регистрации: номер телефона, пароль
  • Данные профиля: дневной лимит зарядки, текущее потребление
  • Данные сессий зарядки: время начала/окончания, потребленная энергия

2.2 Автоматически собираемая информация:

  • Данные устройства: модель телефона, операционная система, версия приложения
  • Данные использования: частота использования, предпочитаемые станции, время зарядки
  • Данные сессий: статус активных сессий зарядки, информация о портах

3. Как мы используем вашу информацию

3.1 Основные цели:

  • Предоставление услуг: управление зарядными сессиями, мониторинг лимитов
  • Улучшение сервиса: анализ использования для оптимизации работы станций
  • Безопасность: предотвращение превышения лимитов и обеспечение безопасности
  • Поддержка: ответы на запросы пользователей и техническая поддержка

3.2 Аналитика и улучшения:

  • Анализ популярности зарядных станций
  • Оптимизация времени ожидания
  • Улучшение пользовательского интерфейса

4. Передача данных третьим лицам

Мы не продаем, не обмениваем и не передаем ваши личные данные третьим лицам, за исключением следующих случаев:

4.1 Поставщики услуг:

  • Облачные сервисы: для хранения данных
  • Аналитические сервисы: для улучшения приложения
  • Сервисы аутентификации: для безопасного входа в систему

4.2 Законные требования:

  • По требованию закона или судебного решения
  • Для защиты наших прав и безопасности
  • В случае слияния или приобретения компании

5. Безопасность данных

5.1 Меры защиты:

  • Шифрование данных при передаче (HTTPS/TLS)
  • Безопасное хранение паролей (хеширование)
  • Регулярные обновления безопасности
  • Ограниченный доступ к персональным данным

5.2 Хранение данных:

  • Данные хранятся на защищенных серверах
  • Регулярное резервное копирование
  • Соблюдение международных стандартов безопасности

6. Ваши права

6.1 Право на доступ:

  • Получить копию ваших персональных данных
  • Узнать, как используются ваши данные
  • Проверить точность информации

6.2 Право на исправление:

  • Исправить неточные данные
  • Дополнить неполную информацию
  • Обновить устаревшие данные

6.3 Право на удаление:

  • Удалить аккаунт и связанные данные
  • Отозвать согласие на обработку данных
  • Ограничить использование данных

6.4 Право на переносимость:

  • Экспортировать ваши данные
  • Перенести данные в другое приложение

7. Дети и несовершеннолетние

7.1 Возрастные ограничения:

  • Приложение предназначено для пользователей от 16 лет
  • Мы не собираем данные детей младше 13 лет
  • При обнаружении данных детей они будут удалены

8. Международная передача данных

8.1 Географическое расположение:

  • Данные могут обрабатываться в разных странах
  • Соблюдение местного законодательства о защите данных
  • Использование стандартных договорных условий ЕС

9. Изменения политики

9.1 Уведомления:

  • Уведомление о значительных изменениях
  • Возможность отозвать согласие
  • Сохранение предыдущих версий политики
